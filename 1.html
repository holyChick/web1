HTML

Hypertext Markup Language (HTML) is the stndard markup language for creating web pages and web applications. Web browsers recieve HTML documents from a web server or form local storage and render them into multimedia web pages. HTML describes the structure of a Webpage semantically and originallu included cues for the appearence of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to creat structed documents quotes and other items. HTML elements are delineated by tags, written using angle by denoting structural semantics for text such as headings, paragraphs, lists, links brackets.
